
## Descrição

Repositório de backend feito com Nest.js. O objetivo da api é ser um CRUD de clientes com integração ao DynamoDB.


## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

```

#### Acesso ao Swagger

```http
:NODE_ENV/api
```
